## SAIB( PC+ 모바일반응형)

### 📋 폴더 구조

- 폴더는 크게 `public`,`src`로 나뉘어져있습니다.
- src 안에 `page폴더`와 `Component폴더`,  `Image폴더` 그리고 `Routes.js파일`이 있습니다.
- page폴더 안에 `Main폴더`가 있습니다.
- component폴더 안에 `Nav폴더`,`Footer폴더`

---
### 사용 스택/Tool  
- HTML
- SCSS
- Git/Github
